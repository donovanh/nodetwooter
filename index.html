Tweets

Embed twitter search results in your static HTML using JavaScript.

How it works

Recently, Twitter made some changes and stopped supporting embedded tweets using pure JavaScript. This was a move to restrict unauthorised users and limit the traffic hitting their servers. We can still use the Twitter API, but since authentication is now required, it's not possible to do so with only JavaScript.

Tweets is a server-side app that provides Twitter search results, with the correct authentication in place. It is currently available as a Node.js app that you host on your own servers, or a free service such as Heroku or Appfog.
